# Python Learning Project

## Overview
This is a Python learning project with examples of object-oriented programming, functional programming, and API requests.

## Recent Changes
**November 6, 2025**
- Fixed indentation errors in `course.py` and `student.py`
- All code is now running without errors
- Configured workflow to run `main.py`

## Project Files
- **student.py**: Student class with grade tracking
- **course.py**: Course class that manages multiple students
- **functional_demo.py**: Examples of list comprehensions and functional programming
- **main.py**: API request example using httpbin.org

## Project Architecture
- Object-oriented design with Student and Course classes
- Test code included at the bottom of student.py and course.py files
- Uses requests library for HTTP requests
